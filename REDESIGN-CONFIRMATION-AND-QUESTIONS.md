## ✅ Confirmed Understanding

### 1. **Rounding Strategy** (UPDATED)
- **NOT always round UP for P**
- Use **midpoint tolerance rule**: Round to nearest practical bag value
- Bag steps: 2.5 kg or 5 kg multiples
- Example: 23.7 kg → midpoint between 22.5 and 25 = 23.75
  - If < 23.75 → round down to 22.5
  - If ≥ 23.75 → round up to 25
- Slight under/over application acceptable, but must respect overall tolerance

### 2. **K Threshold (35-40% Rule)**
- If complex fertilizer supplies **35-40% of stage K requirement** → Don't add MOP
- If only **~30%** → Acceptable, shift remaining to later stages
- Missing K should be shifted to Stage 3 (not forced in same stage)
- Overall K must still reach ~100% within 1-2% tolerance

### 3. **Excess Tolerance (Two-Level)**
- **Stage level**: Up to ±10% deviation allowed
- **Overall level**: Maximum 1-2% excess only
- Stage deviations must NOT accumulate
- Final total N, P, K must be ~100% of recommendation

### 4. **K Deviation Acceptance**
- Stage-wise distribution can vary (30% instead of 50% is OK)
- Overall K must reach 100% within 1-2% tolerance
- Adjust later stages to compensate

### 5. **Product Minimization**
- **Maximum 3 products per stage**
- Prefer reusing fertilizers across stages
- But only if it matches soil test data

### 6. **SSP Usage**
- Only basal/early stages
- Never Stage 3
- P must be completed by Stage 1 or 2

### 7. **Stage 3 P Handling**
- **Never add P to Stage 3**
- If P deficit detected → Pull back to Stage 1 or 2
- Never push P into Stage 3

### 8. **Core Principle**
- **P-first selection**: Always start with P, select complex fertilizer
- Then adjust N (with urea)
- Then fine-tune K (with MOP if needed)

### 9. **Soil Imbalance Rules**
- Don't add nutrients that soil already has in excess
- System must warn if fertilizer worsens imbalance
- Example: If P is high, don't select high-P fertilizers
